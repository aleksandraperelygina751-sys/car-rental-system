# Система проката автомобилей

Курсовой проект по базам данных.  
Приложение для автоматизации работы компании по прокату автомобилей.


##  Возможности

- Авторизация и регистрация пользователей
- Три роли: ADMIN, MANAGER, USER
- Просмотр, добавление, изменение и удаление автомобилей
- Просмотр, добавление, изменение и удаление клиентов
- Создание договоров аренды с автоматическим расчётом стоимости
- Автоматическое применение скидок (в зависимости от количества аренд)
- Просмотр и добавление штрафов
- Личный кабинет пользователя


##  Требования для запуска

- Java 17 или выше
- JavaFX SDK 21
- MySQL Server (или XAMPP с MySQL)
- База данных `car_rental`


##  Как запустить

### 1. Установить базу данных

1. Запустите XAMPP (MySQL)
2. Откройте phpMyAdmin (http://localhost/phpmyadmin)
3. Создайте базу данных `car_rental`
4. Выполните SQL-скрипт из файла `car_rental.sql` (он есть в репозитории)

### 2. Запустить приложение
Откройте командную строку в папке с JAR-файлом и выполните:

```bash
java --module-path "путь_к_папке_javafx/lib" --add-modules javafx.controls,javafx.fxml -jar car-rental-system.jar
